Project

Pengalaman Pembelajaran Jarak Jauh

Pada kesempatan kali ini, saya akan membagikan pengalaman saya perihal pembelajaran jarak jauh yang sudah lebih dari 1 tahun saya jalani. Pada awalnya, saya tentu merasa kesulitan untuk mencerna pembelajaran karena jujur saya termasuk individu yang sulit mengatur waktu dan sukar untuk belajar mandiri. di awal-awal pandemi saya hanya belajar saat mengerjakan tugas, itupun tugasnya sering menunggu pekerjaan punya teman selesai agar saya tahu bagaimana cara mengerjakannya. Tapi untungnya saya masih bisa mengerti walaupun harus berkali-kali bertanya kepada teman saya, semoga teman saya tidak menyimpan dendam ke saya. Selain itu, saya juga sering keteteran dalam mengerjakan tugas. Sebenarnya tugasnya tidak menyusahkan, hanya saja saya jarang mengecek gadget. Jadi, saat saya melihat adanya tugas, ternyata tenggat pengumpulannya sudah mepet. Atau kadang, saya sudah menaruh tugas di kantung tugas, tapi saya lupa untuk mengumpulkannya, sehingga jadinya tercatat telat. Dan itu seringkali terjadi sampai sekarang. ampai pada sekarang, tentu saja banyak keluhan yang saya kumpulkan karena pebelajaran jarak jauh. Terkadang media penunjang pembelajaran jarak jauh saya sering bermasalah sehingga saya kesal sendiri jadinya. Tatap muka secara tidak langsung melalui zoom meeting dan kondisi belajar yang terpaksa dilakukan dirumah seringkali membuat saya jenuh. Karena saat saya ada dirumah, saya merasa rumah adalah tempat saya untuk beristirahat, bukan untuk lainnya.

Jadi saya merasa sulit untuk mengatur mood saya agar semangat sekolah. Selain itu, untungnya sekolah saya beberapa kali menyediakan pulsa gratis untuk muridnya, sehingga saya tidak perlu mencemaskan perihal kuota belajar. Tapi banyak juga hal yang saya dapatkan selama pembelajaran jarak jauh ini. Pertama, saya jadi lumayan mahir menggunakan gadget, maksudnya, selama pandemi ini sekolah banyak mengajarkan untuk mengerjakan tugas dengan berbagai macam aplikasi. Lalu, saya juga merasa menjadi lebih dekat dengan anggota keluarga saya terutama kakak kakak saya.Saya sangat berharap pandemi ini cepat berakhir. Sehingga semua bisa menjalankan aktivitasnya lebih leluasa seperti sebelumnya, sehingga saya bisa sekolah bertatap muka dan melepas rindu dengan teman teman saya, sehingga kami para siswa bisa mencerna pelajaran lebih baik dan semakin baik daripada sebelumnya.Sampai disini dulu ya tulisan saya kali ini, semoga semua keresahan kita cepat memudar dan kita senantiasa diberi kesehatan oleh Yang Maha Esa. Terima kasih banyak sudah membaca tulisan saya seputar pengalaman yang nyatanya lebih meyakinkan jika disebut keluhan. Sampai jumpa lagiii!!
